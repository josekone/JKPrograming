Por que tener un sitio web para mi negocio?



Actualmente los usuarios y potenciales clientes buscan referencias sobre productos y servicios en el mayor motor de búsqueda en el mundo, Google. Para estar indexado a Google necesitas una pagina Web, presentable, alojada en un buen servidor y de buena calidad, para proporcionarles a tus posibles clientes confianza al relacionar tu empresa.

Con un sitio Web,los usuarios podran acceder a la informacion de tu negocio y tu empresa solo con contar con una conexion a internet. Asi podran obtener tus servicios, contactarte.Con un buen sitio web el nivel de clientes potenciales se elevará considerablemente.

Diferénciate dentro de tu sector con una página sencilla, fresca y nueva. Dale credibilidad y añade valor con un contenido siempre actualizado y de interés para tus clientes. Eso potenciará los motores de búsqueda para que te encuentren más fácilmente.

Una página web es una manera de publicitarnos diariamente con un mínimo mantenimiento. Un anuncio permanente que puede contar diferentes cosas según lo vayamos adaptando a las necesidades cambiantes de nuestro público y que tiene un alcance mundial con un coste realmente bajo.

Una página Web, es como tener una oficina de nuestra empresa abierta al público en general las 24 horas y 365 días al año ofreciendo información importante a clientes potenciales que buscan lo que ofrecemos. Asi que no lo pienses mas y comienza a generar nuevos clientes.

Tecnologias?

Tecnologías que forman parte del desarrollo sobre el entorno Web.

El diseño y desarrollo de aplicaciones web consiste en implementar sus necesidades, objetivos o ideas en Internet utilizando las tecnologías más idóneas según su proyecto. Las aplicaciones web ofrecen servicios a los usuarios de Internet que acceden utilizando un navegador web.

CSS podría definirse como un tipo de lenguaje que permite definir y crear la presentación de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML.

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

JavaScript es el lenguaje de programación que debes usar para añadir características interactivas a tu sitio web.

Sass permite el uso de dos sintaxis diferentes para crear sus archivos. La primera, conocida como SCSS (del inglés, Sassy CSS) es la que se utiliza en este manual y es una extensión de la sintaxis de CSS3.
